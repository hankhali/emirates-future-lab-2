# Emirates Future Lab

Interactive Grade 4 learning experience focused on sustainability, AI, and UAE-themed missions.

## Quick Start

1. Install dependencies:
   npm install
2. Run the app:
   npm run dev
3. Open the local URL shown in the terminal (usually http://localhost:3000).

## Demo Login and Sign Up (For Review)

This project is set up for demo/testing. Reviewers can use dummy values.

- Full Name: any test name
- Email: use any gmail.com address
- Password: any test password

Example values:

- Full Name: Test Student
- Email: test.student4@gmail.com
- Password: Test1234

You can also use Sign In mode with dummy values, for example:

- Email: reviewer.demo@gmail.com
- Password: DemoPass2026

## Notes For Manager Review

- No real student data is required for testing.
- Use fake names and gmail.com test emails only.
- Grade and subject are prefilled in the form flow for this demo.

## Scripts

- npm run dev: start development server
- npm run build: production build (client + server bundle)
- npm run start: run production server
- npm run check: TypeScript type check
- npm run preview: preview built client
